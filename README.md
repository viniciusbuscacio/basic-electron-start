# basic-electron-start
Basic Electron.js boilerplate for my personal projects. 

## Features
- [x] Electron.js
- [x] Electron-builder
- [x] Javascript
- [x] HTML
- [x] CSS
- [x] Bootstrap


# how to use
```bash
git clone https://github.com/viniciusbuscacio/basic-electron-start.git
cd basic-electron-start
npm install
npm start
```
